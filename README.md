## ✨ Key Features
- **User-Friendly Interface:** Intuitive navigation for both donors and receivers.
- **Real-time Tracking:** Efficiently track donation status.
- **Contribution Dashboard:** Donors can manage and monitor their donation history.

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB / SQL

## 🚀 How to Run Locally
1. Clone the repository: `git clone https://github.com/CodeNinja47-star/Food-Donation-platform.git`
2. Navigate to the project directory: `cd Food-Donation-platform`
3. Install dependencies: `npm install`
4. Start the application: `npm start`
